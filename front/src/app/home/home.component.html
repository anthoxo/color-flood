Competition Goal Having the most colored tiles at the end of the game. How to play ? At each turn, you can color your zone to a different color to extend it. Be cautious You are not alone in the grid ! Someone else would have the same goal as you. Best of 5 You need to win 3 plays to win the game. Program inputs Start Size of the grid Each turn Matrix which represents the state of the grid when it is your turn. Colors Between 5 and 20 different colors. In the code, each color is represented with a number between 0 and 19. Authorized languages Java, TypeScript, Python.
